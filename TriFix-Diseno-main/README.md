# TriFix-Diseno
Version 1.0
Ultima actualizacion 6 de Noviembre de 2025
Repositorio del diseño de nuestro proyecto TriFix.
Repositorio en el cual se presentaran el diseño de las diferentes interfaces que le daran vida a nuestro proyecto, interfaces como:
*Ventana inicial: Le da un primer vistazo al usuario de lo que sera la pagina.
*InicIar secion: Ventana en la cual el usuario podra colocar su usuario y contraseña.
*Registro de cuentas: Ventana en la cual el usuario podra registrarse en caso de no contar con una cuenta.
*Dashboard: Ventana en la cual el ususario podra comenzar a poner a prueba la funcionalidad del sitio.
