# TriFix-DesarrolloWeb
Version 1.0
Ultima actualizacion 6 de Noviembre de 2025
Repositorio del desarrollo web de nuestro proyecto TriFix.
En este repositorio se guarda parte del desarrollo de la aplicacion, aunque todavia es una parte no funcional del propio archivo, se espera que en una futura entrega se le pueda estar dando mantenimiento a la aplicacion y que esta pueda funcionar en su totalidad, falta mucho todavia para llegar a este proceso pero segun la ultima reunion de nuestro equipo nos pusimos de acuerdo segun las normas SCRUM para que cada quien pueda liderar una parte del proyecto determinado a su puesto que le toco.
